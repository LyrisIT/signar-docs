# Versión 12.6

Una nueva version se encuentra disponible para Signar. A continuación presentamos los cambios introducidos:

## Changelog 

- SIGNAR-380 Se agregaron metodos de validacion sobre los recibos que un empleador sube con el firmador para evitar que se suban archivos vacios, mal formados o invalidos.
- SIGNAR-354 Mejoras de seguridad en cuestiones de inicio de sesión, para brindar mejor monitoreo de actividad.
